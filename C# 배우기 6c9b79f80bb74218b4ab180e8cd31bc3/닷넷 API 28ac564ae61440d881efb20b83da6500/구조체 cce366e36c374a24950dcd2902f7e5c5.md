# 구조체

---

- 닷넷 API에서 제공하는 클래스들을 정리한 페이지.

## Math 클래스

---

- 닷넷에서 제공하는 수학 관련 내장 클래스이다.
- Math.E : 자연 로그 상수 값 출력
- Math.PI : 원주율 출력
- Math.Abs() : 절댓값 출력
- Math.Max() : **최댓값 출력**
- Math.Min() :  **최솟값 출력**
- Math.Pow() : **제곱근 출력**
- 위의 명령어 외에도 다양한 메서드들이 있다.
- 네임스페이스에 **using static System.Math;** 를 입력하면 명령어를 줄일 수 있다.
